# claudecode

Claude Code を使って学ぶためのサンドボックスリポジトリです。

## このリポジトリについて

このリポジトリは Claude Code の練習・実験用として作成しました。
コードの変更、PR の作成、レビューの流れを体験することを目的としています。

## はじめかた

1. リポジトリをクローン
   ```bash
   git clone https://github.com/Yuna780/claudecode.git
   ```
2. フォルダに移動
   ```bash
   cd claudecode
   ```

## コントリビューション

1. ブランチを作成する (`git checkout -b feature/your-feature`)
2. 変更をコミットする (`git commit -m "Add some feature"`)
3. ブランチをプッシュする (`git push origin feature/your-feature`)
4. Pull Request を開く
